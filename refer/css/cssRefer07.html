ir효과

ir효과는 CSS을 통하여 대체문자 관리하는 방법입니다.

ir_pm

ir_pm는 의미있는 이미지에 대체 텍스트를 제공하는 경우을 의미합니다.

사용방법

ir_pm {
    display: block;
    overflow: hidden;
    font-size: 0;
    line-height: 0;
    text-indent: -9999px;
}

ir_wa

ir_wa는 의미있는 이미지에 대체 텍스트로 이미지가 없어도 대체 텍스트를 보여주고 싶은 경우를 의미합니다.

사용방법

.ir_wa {
    display: block;
    overflow: hidden;
    position: relative;
    z-index: -1;
    width: 100%;
    height: 100%;
}

ir_so

ir_so는 대체 텍스트가 아닌 접근성을 위한 숨김 텍스트를 제공하는 경우를 의미합니다.

사용방법

.ir_so {
    overflow: hidden;
    position: absolute;
    width: 0%;
    height: 0%;
    line-height: 0;
    text-indent: -9999px;
}
