단위 이름 값
cm 센티미터(Centimeters) 1cm = 37.8px = 25.2/64in

in 인치(Inches) 1in = 2.54cm = 96px
pc 파이카(Picas) 1pc = 1/6th of 1in
pt 포인트(Points) 1pt = 1/72th of 1in
px 픽셀(Pixels) 1px = 1/96th of 1in

상대 크기 단위

단위 설명
em 자식 요소들의 폰트 사이즈에 영향을 줍니다.
ex pass 요소 글꼴의 x-height.
ch pass 요소 글꼴의 glyph "0" 의 사전 길이 (너비) 입니다.
rem pass 루트 요소의 글꼴 크기.
lh pass 요소의 라인 높이.
vw pass viewport 너비의 1%.
vh pass viewport 높이의 1%.
vmin pass viewport 의 작은 치수의 1%.
vmax pass viewport 의 큰 치수의 1%.
