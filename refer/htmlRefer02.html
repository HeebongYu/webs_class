웹표준(Web standards)

  • 웹 표준이란 브라우저 종류 및 버전에 따른 기능 차이에 대하여 호환이 가능하도록 제시된 표준입니다.
  • 표준화 단체인 W3C(World Wide Consortium) 권고한 표준안을 기준으로 합니다.
  • HTML, CSS, JavaScript 등에 대한 규정을 담고 있습니다.
  • 목적은 웹사이트에 접속한 사용자가 어떠한 운영체제나 브라우저를 사용하더라도 동일한 결과를 보이게 하는 것입니다.

    웹접근성(Web Accessibility)

    • 신체 및 환경의 제약에 얽매이지 않고 웹 사이트에서 제공하는 정보에 접근해 이용할 수 있도록 하는 것을 의미합니다.
    • 신체 제약으로 시각장애인에게는 시각 정보에 대한 설명, 청각장애인에게는 음성 정보의 자막 제공, 지체장애인에게는 키보드만으로 쓸 수 있도록 하는 것 입니다.
    • 환경 제약으로 비표준화된 특정 기술을 쓰지 않고 표준만 준수하면 어떠한 장애 없이 웹을 쓸 수 있도록 해야 합니다.
    • 웹 접근성을 고려한 콘텐츠 제작 방법은 쉬운 인식, 쉬운 운용, 쉬운 이해, 견고함을 기반으로 합니다.

    웹호환성(Cross Browsing)

    • 웹 호환성은 표준 웹 기술을 사용하여 운영체제, 브라우저 등 어느 한쪽으로 최적화되거나 종속되지 않도록 공통 요소를 사용하여 웹 페이지를 제작하는 기법입니다.
    • 웹 사이트 사용 시 운영체제 및 브라우저 간 동일한 결과가 나오도록 의미하는 웹 상호운용성의 개념에 웹 표준의 준수를 포함하는 개념입니다.
