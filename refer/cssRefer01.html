CSS란?

  • CSS는 마크업 언어(HTML, XHTML)가 실제 표시되는 방법을 기술하는 스타일 언어입니다.
  • 레이아웃과 스타일을 정의할 때 자유도가 높습니다.

CSS 주석 표시

  • CSS는 주석을 표시할 때 시작과 끝에 /* 주석 */를 넣습니다.

/*
    css 주석 부분입니다.
*/

CSS 선언 방법

내부 스타일(Internal Style Sheet)

  • HTML 문서 안의 <style>과 </style> 안에 CSS 코드를 넣는 방법입니다.

※HTML
    
<style>
    h1 {
        color: blue;
    }
</style>

외부 스타일(Linking Style Sheet)

  • 별도의 CSS 파일을 만들고 HTML 문서와 연결하는 방법입니다.

※HTML

<link rel="stylesheet" href="css/style.css">


※CSS

h1 {
    color: red;
}

인라인 스타일(Inline Style Sheet)

  • HTML 태그의 style 속성에 CSS 코드를 넣는 방법입니다.

※HTML

<p style="color: blue">Lorem ipsum dolor. </p>
