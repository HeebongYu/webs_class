자바스크립트 변수

JavaScript 변수를 선언하는 4가지 방법

  • var 사용
  • let 사용
  • const 사용
  • 아무것도 사용하지 않음

변수란 무엇입니까?

변수는 데이터를 저장하기 위한 컨테이너입니다(데이터 값 저장).

JavaScript 변수는 언제 사용합니까?

항상 var, let 또는 const를 사용하여 JavaScript 변수를 선언 하십시오.
var키워드는 1995년부터 2015년까지 모든 JavaScript 코드에 사용되었습니다.
let 및 const키워드는 2015년에 JavaScript에 추가되었습니다.
이전 브라우저에서 코드를 실행하려면 var를 사용해야 합니다.
