String.prototype.toUpperCase() : 메서드 : 문자열을 대문자로 변환하여 반환합니다.

const str = "javascript";
console.log(str.toUpperCase());
//JAVASCRIPT

String.prototype.trim() : 메서드 : 문자열 양 끝의 공백을 제거하여 반환합니다.

const str = " javascript ";
console.log(str);
// javascript
console.log(str.trim());
//javascript

String.prototype.indexOf() : 메서드 : 주어진 값과 일치하는 첫 번째 인덱스를 반환합니다.

const str = "javascript";
console.log(str.indexOf("a"));
//1
console.log(str.indexOf("i"));
//7 console.log(str.indexOf("b"));
//-1

String.prototype.lastIndexOf() : 메서드 : 주어진 값과 일치하는 뒤에서 첫 번째 인덱스를 반환합니다.

const str = "javascript";
console.log(str.lastIndexOf("a"));
//3
console.log(str.lastIndexOf("i"));
//7
console.log(str.lastIndexOf("b"));
//-1

String.prototype.search() : 메서드 : 정규식 표현이 사용 가능합니다. indexOf()와 반환방식은 동일합니다.

console.log(str.search(/a/));
//1
console.log(str.search(/i/));
//7
console.log(str.search(/b/));
//-1
