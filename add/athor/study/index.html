프론트엔드 웹 개발자 학습 경로에 오신 것을 환영합니다!

여기에서 우리는 프론트 엔드 웹 개발자가 되기 위해 알아야 할 모든 것을 가르쳐줄 구조화된 과정을 제공합니다.

각 섹션을 진행하면서 새로운 기술을 배우거나 기존 기술을 항상시키십시오

각 섹션에는 다음 단계로 진행하기 전에 이해도를 테스트라기 위한 연습과 평가가 포함되어 있습니다.

  • CSS: 웹 페이지 스타일 지정에 사용되는 언어
  • JavaScript: 웹에서 동적 기능을 생성하는 데 사용되는 스크립팅 언어
