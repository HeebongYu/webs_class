자바스크립트(JavaScript)

자바스크립트는 복잡한 무언가를 웹페이지에 적용할 수 있게 하는 스크립트 혹은 프로그래밍 언어입입니다.

HTML은 웹의 내용을 작성하고 CSS은 웹을 디자인하다면 JavaScript는 웹의 동작합니다.

  • 자바스크립트는 변수안에 값을 저장할 수 있습니다.
  • 자바스크립트는 문자들을(HTML, CSS 작성내용) 조작 가능합니다.
  • 자바스크립트는 이벤트에 응답하는 코드를 작성할 수 있습니다.

자바스크립트 기본문법

기본규칙

  • 실행문은 세미콜론(;)으로 구분됩니다.
  • 대·소문자를 구분하여 작성합니다.
  • 문자형 데이터 작성할 때에는 " "와 ' '의 겹침 오류를 주의합니다.
  • 코드를 작성할 떄 {} 또는 ()의 짝이 맞아야 합니다.

주석(comment)

  • 선언문 안에 코드가 아닌 설명글을 넣고 싶으면 주석처리를 합니다.
  • 한 줄 주석인 경우에는 '//한 줄 설명글'로 작성합니다.
  • 여러 줄 주석인 경우에는 '/*여러 줄 설명글*/'로 작성합니다.
{
    //한 줄 설명글인 경우
    /*
        설명글이 여러 줄인 경우
        이렇게 처리합니다.
    */
}

리터럴(literal)

리터럴은 직접 표현되는 값 그 자체를 의미합니다.

자료형(Data type)으로 다양한 종류가 있습니다.

여기서는 자료형의 기본적인 형태만 다루고 따로 살펴보겠습니다.

  • 숫자형(Number)
    • {
    012346789
}
  • 문자형(String)
    • {
    "a", "b", "c", "abc", "1", "2", "3", "123"
}
  • 논리형(Boolean)
    • {
    true, false
}
  • Null / Undefined / Typeof
    • {
    null, undefined, typeof
}
  • 배열(Array)
    • {
    [0, 1, "0", "a"];
}
  • 객체(Object)
    • {
    {0, 1, "0", "a"};
}
  • 함수(Function)
    • {
    function name();
}
  • 심볼(Symbol)
    • {
    Symbol();
}
