var

  • 데이터 저장 가능
  • 중복선언 가능
  • 재활당(변경) 가능

중복선언 가능

코드

{
    var a = "var 1";
    var a = "var 2";
    
    document.write(a);
}

결과

재활당(변경) 가능

코드

{
    for(var b=1; b<5; b++){
        document.write(b)
    }
}

결과
