JavaScript 기본

JavaScript는 여러분이 인터렉티브한 기능을 웹사이트에 추가할 때 사용하는 프로그래밍 언어입니다. 예를 들면 게임, 버튼이 눌리거나 품에 정보가 입력되었을 떄 발생하는 것, 동적인 스타일 효과, 애니매이션 등이 있습니다.JavaScript 기본은 어려분께 흥미로운 언어로 가능할 것들에 대한 것과 어떻게 시작할 수 있는지에 대한 아이디어를 제공할 것입니다.

그래서 JavaScript가 뭔가요?

JS(JavaScript)는 HTML 문서에 적용될 때 웹사이트상에서 동적 상호작용성을 제공할 수 있는 프로그래밍 언어입니다. 캐러셀 기능(이미지를 차례대로 바꿔볼 수 있는 기능)을 가진 이미지 갤러리, 변화하는 레이아웃, 그리고 버튼이 클릭될 때의 반응과 같은 간단한 것부터 시작할 것입니다. 경험이 더 쌓이면 게임이나 움직이는 2D 및 3D 그래픽, 포괄적인 데이터베이스 지향적인 앱과 더 많은 것을 만들 수 있을 것입니다!

  1. scripts 폴더에 main.js로 파일을 저장하세요.
  2. index.html 파일에 </body> 태그 바로 앞에 아래 코드를 붙여 넣으세요. 
    <iscript src="scripts/main.js"><i/script>
  3. main.js 파일에 다음 코드를 추가하세요. 
    let myHeading = document.querySelector('h1');
myHeading.textContent = 'Hello world!';

제목이 Hello world! 변경되었다면 성공한 것입니다.

  • let myHeading = 이것은 let을 이용하여 변수 myHeading를 만든것입니다.
  • document.querySelector('h1'); 이것은 querySelector()을 이용하여 <h1>를 가져온것입니다.
  • let myHeading = document.querySelector('h1'); 이것은 myHeading에 <h1>을 저장한 것입니다.
  • myHeading.textContent = 'Hello world!'; 이것은 textContent을 이용하여 myHeading(<h1>)의 내용을 Hello world!로 설정한 것입니다.

언어 기본 특강

이 모든 것이 작동하는 방식에 대해 더 잘 이해할 수 있도록 자바스크립트 언어의 핵심적인 특징을 몇 가지 설명드리겠습니다. 이런 특징은 모든 프로그래밍 언어에서 공통적으로 나타나는 것임으로 이런 원칙을 숙지한다면 어떤 것이라도 프로그램할 수 있게 될 것입니다.

변수(Variables)

변수는 여러분이 값을 저장할 수 있는 컨테이너입니다. 변수를 선언할 때 const 또는 let 키워드 뒤에 원하는 어떤 이름을 붙이면 됩니다.

  • 변수를 선언한 후 값을 할당할 수 있습니다. 
    let myVariable;
myVariable = 'Bob';
  • 변수 선언과 값을 주는 작업을 한 줄로 처리 할 수 있습니다. 
    let myVariable = 'Bob';
  • 이름으로 변수를 호출하기만 하면 값을 추출할 수 있습니다. 
    myVariable;
  • 변수에 어떤 값을 준 후 나중에 변경할 수 있습니다. 
    let myVariable = 'Bob';
myVariable = 'Steve';

자세한 내용는 JavaScript의 타입과 자료구조에서 선행해도 좋습니다.

자료형(Data Types)

  • 문자열(String) : 문자열이라는 것을 나타내기 위해서는 인용부호("" 또는 '')로 둘러싸야 합니다.
    구문 : let myVariable = 'Bob';
  • 숫자(Number) : 숫자는 인용부호를 사용하기 않습니다.
    구문 : let myVariable = 10;
  • 불린(Boolean) : 참(true)/거짓(false)을 말하며 인용부호가 필요 없습니다.
    구문 : let myVariable = true;
  • 배열(Array) : 여러 값을 하나의 단일 참조에 저장할 수 있도록 하는 구조입니다.
    구문 : let myVariable = [1,'Bob','Steve',10];
  • 객체(Object) : 기본적으로 무엇이든 자바스크립트의 객체이며 어떤 변수에 저장될 수 있습니다.
    구문 : let myVariable = document.querySelector('h1');
    위의 모든 예시도 해당됩니다.

자세한 내용는 JavaScript의 타입과 자료구조에서 선행해도 좋습니다.

연산자(Operator)

연산자는 두 값(또는 변수)로부터 결과를 만들어내는 수학 기호입니다.

  • 더하기 : +
  • 빼기 : -
  • 곱하기 : *
  • 나누기 : /
  • 할당 : =
  • 동등 : ==, ===
  • 부정 : !
  • 다름 : !==, !===

자세한 내용는 표현식과 연산자에서 선행해도 좋습니다.

조건문

조건문은 어떤 표현식(expression)이 참인지 아닌지를 테스트하고 그 결과에 따라 선택적으로 코드를 실행할 수 있도록 하는 코드 구조입니다.
구문 : if(조건식){실행문}

함수(Functions)

함수는 재사용하기를 원하는 기능을 담는 방법입니다. 필요할 때 매 번 전체 코드를 다시 작성하는 대신 함수의 이름으로 그 함수를 호출할 수 있습니다.

자세한 내용는 JavaScript의 함수에서 선행해도 좋습니다.

이벤트

웹사이트의 실질적인 상호작용에는 이벤트가 필요합니다. 이벤트는 브라우저에서 발생하는 일을 듣고 그에 대한 반응으로 코드를 실행하는 코드 구조입니다.

이미지 변경자 추가하기

이미지를 하나 더 추가하고, 이미지가 클릭될 때 자바스크립트를 이용해 두 이미지 사이에 전환이 이루어지도록 해보겠습니다. 

let myImage = document.querySelector('img');

myImage.onclick = function() {
    let mySrc = myImage.getAttribute('src');
    if(mySrc === 'images/img01.png') {
      myImage.setAttribute ('src','images/img02.png');
    } else {
      myImage.setAttribute ('src','images/img01.png');
    }
}
  • let myImage = document.querySelector('img'); : 이것은 변수(myImage)를 만들고 변수에 객체(img)을 문서에서 찾아(document.querySelector()) 저장한 것입니다.
  • myImage.onclick = function(){} : 이것은 변수(myImage)에 대하여 클릭이벤트(.onclick)가 발생하면 함수(function)를 실행하는 것입니다.
  • let mySrc = myImage.getAttribute('src'); : 이것은 변수(mySrc)에 변수(myImage)의 속성(getAttribute())이 src인 것의 값은 저장한 것입니다.
  • if(mySrc === 'images/img01.png'){} :이것은 조건문(if)으로 변수(mySrc)가 images/img01.png와 같은때 실행됩니다.
  • else {} 것은 앞의 조건문에 만족하지 않는 경우 실행됩니다.
  • myImage.setAttribute ('src','images/img02.png'); : 이것은 변수(myImage)의 속성(src)의 값을 images/img02.png으로 변경합니다.

개인화된 환영 메시지 추가하기

다음으로는 사용자가 처음으로 사이트에 방문했을 때 페이지의 제목을 개인화된 환영 메시지로 바꾸는 것입니다. 이 메시지를 웹 저장소 API(Web Storage API)를 이용해 저장할 것입니다. 사용자를 변경할 수 있는 옵션도 추가하여 언제든지 필요할 때 환영 메시지를 나타나도록 할 것입니다.

  1. index.html 파일에 <script> 바로 앞에 아래 코드를 적습니다. 
    <button>Change user</button>
  2. main.js 파일의 맨 마지막에 아래 코드를 추가합니다. 
    let myButton = document.querySelector('button');
let myHeading = document.querySelector('h1');
  3. 개인화된 인삿말을 설정하기 위해 다음 함수를 추가합니다. 다만 아직 아무 동작도 하지않지만 좀 이따가 고칠겁니다. 
    function setUserName() {
    let myName = prompt('Please enter your name.');
    localStorage.setItem('name', myName);
    myHeading.textContent = 'Mozilla is cool, ' + myName;
}
