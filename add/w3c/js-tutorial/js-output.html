자바스크립트 출력

JavaScript는 다양한 방식으로 데이터를 표시할 수 있습니다.

HTML 요소의 변경 - innerHTML 사용하기

예시 
<p id="demo"></p>
<script>
    document.getElementById("demo").innerHTML = 5 + 6;
</script>
  • innerHTML 속성은 HTML에 요소의 데이터를 변경하는 일반적인 방법입니다.
  • ~.innerHTML = 5 + 6; : ~에 HTML 요소를 "5 + 6"으로 변경합니다.

단순 입력(테스트용) - document.write() 사용하기

예시 

<script>
    document.write(5 + 6);
</script>
  • document.write() 메서드는 테스트용으로만 사용해야 합니다.
  • document.write(5 + 6); : 하단에 "5 + 6"을 입력합니다.
