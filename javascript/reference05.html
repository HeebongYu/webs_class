연산자(operators)

  • 연산자에는 산술, 문자 결합, 대입, 증감, 비교, 논리 연산자가 있습니다.
  • 연산자는 조건식을 작성할 떄 사용합니다.

산술 연산자

  • 산술 연산자는 더하기(+), 빼기(-), 곱하기(*), 나누기(/), 나머지(%)가 있습니다.
A 산술연산자(+, -, *, /, %) B 
{
    let i = 10;
    let j = 2;

    document.write(i + j);    //10 + 2 = 12
    document.write(i - j);     //10 - 2 = 8
    document.write(i * j);     //10 * 2 = 20
    document.write(i / j);     //10 / 2 = 5
    document.write(i % j);     //10 % 2 =0
}

문자 결합 연산자

  • 피연산자(연산 대상 데이터)가 문자형 데이터 입니다.
  • 여러 개의 문자를 하나의 문자형 데이터로 결합할 때 사용합니다.
A + B + C 
{
    let t1 = "강";
    let t2 = "아";
    let t3 = "지";

    document.write(t1 + t2 + t3);    //강 + 아 + 지 = 강아지
}

대입 연산자

  • 대입 연산자(=)는 연산된 데이터를 변수에 저장할 때 사용합니다.
  • 복합 대입 연산자(+=, -=, *=, /=, %=)는 산술 연산자와 대입 연산자가 복합적으로 적용된 것입니다.
A 대입연산자(+=, -=, *=, /=, %=) B 
{
    let i = 10;
    let j = 2;

    document.write(i = i += j);   //i = 10 + 2 = 12
    document.write(i = i -= j);   //i = 12 - 2 = 10
    document.write(i = i *= j);   //i = 10 * 2 = 20
    document.write(i = i /= j);    //i = 20 / 2 = 10
    document.write(i = i %= j);   //i = 10 % 2 = 0
}

증감 연산자

  • 증감 연산자는 증가 연산자(++)와 감소 연산자(--)가 있습니다.
  • 증가 연산자(++)는 숫자형 데이터를 1씩 증가시키는 연산자입니다.
  • 감소 연산자(--)는 숫자형 데이터를 1씩 감소시키는 연산자입니다.

비교 연산자

  • '크다, 작다, 같다'와 같이 비교할 때 사용하는 연산자입니다.
  • 결과값으로 trum(참) 또는 false(거짓)로 반환합니다.
종류 설명
A>B A가B 보다 크다.
A<B A가 B보다 작다.
A>=B A가 B보다 크거나 같다.
A<=B A가 B보다 작거나 같다.
A==B A와 B는 같다.
A!=B A와 B는 다르다.

논리 연산자

  • 논리 연산자는 ||(or), &&(and), !(not)가 있습니다.

연산자 우선순위

  • 1. ()
  • 2. 단항 연산자(--, ++, !)
  • 3. 산술 연산자(+, -, *, /, %)
  • 4. 비교 연산자(>, <, >=, <=, ==, !=)
  • 5. 논리 연산자(||, &&)
  • 6. 대입 연산자(=, +=, -=, *=, /=, %=)
