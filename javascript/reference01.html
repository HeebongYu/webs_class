자바스크립트 선언문

  • 선언문은 자바스크립트 코드를 작성할 영역을 선언하는 것입니다.
  • <script> 태그로 선언문이 시작하여 </script> 태그로 선언문이 종료됩니다.
  • <script>부터 </script>까지를 스크립트 영역이라고 부릅니다.
  • 선언문은 <head> 영역 또는 <body> 영역에 선언합니다.
<script>
    자바스크립트 코드;
</script>

자바스크립트 선언문 실습해보기

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <script>
        document.write("환영합니다.")
    <script>
</head>
<body>
</body>    
</html>

자바스크립트 주석처리

  • 선언문 안에 코드가 아닌 설명글을 넣고 싶으면 주석처리를 합니다.
  • 한 줄 주석인 경우에는 '//한 줄 설명글'로 작성합니다.
  • 여러 줄 주석인 경우에는 '/*여러 줄 설명글*/'로 작성합니다.
//한 줄 설명글인 경우
/*
    설명글이 여러 줄인 경우
    이렇게 처리합니다.
*/

코드 입력 시 주의사항

  • 대·소문자를 구분해야 작성합니다.
  • 작성한 후에는 세미콜론(;)을 작성합니다.
  • 한 줄에 한 문장만 작성하도록 합니다.
  • 문자형 데이터 작성할 때에는 " "와 ' '의 겹침 오류를 주의합시다.
  • 코드를 작성할 떄 {} 또는 ()의 짝이 맞아야 합니다.
