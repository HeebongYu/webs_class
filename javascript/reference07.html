반복문(Repetitive Statement)

  • 반복문은 이용하면 코드를 원하는 횟수만큼 반복하여 실행할 수 있습니다.
  • 반복문은 종류로는 while문, do while문, for문, breack문, continue문이 있습니다.

while문

  • while문은 조건식을 만족할 때까지 코드를 여러 회 반복하여 실행합니다.
let 변수 = 초기값; while( 조건식 ){ 자바스크립트 코드; 증감식 } 
{
    let i = 1;
    while( i <= 4 ){ document.write( i ); i++ }  //1234
}

do while문

  • do while문은 반드시 한 번은 코드를 실행하고 조건식 검사합니다.
let 변수 = 초기값; do{ 코드; 증감식 } while( 조건식 ); 
{
    let i = 1;
    do { document.write( i ); i++; } while ( i <= 4 );  //1234
}

for()문

  • for 문은 조건식을 만족할 때가지 특정 코드를 반복하여 실행합니다.
  • 초기값 → 조건식 → 자바스트렙트 코드 → 증감식 → 조건식
for( 초기값; 조건식; 증감식 ){ 자바스크립트 코드; } 
{
    for( i=1; i<10; i++ ){ document.write(i); }    //123456789
}

중첩 for문

  • 중첩 for문은 반복문 안에 반복문이 있는 경우를 의미합니다.
for( 조건식1 ){ if( 조건식2 ){ 자바스크립트 코드; } } 
{
    for( let i=1; i<=2; i++ ){ document.write(i)
        for( let j=1; j<=3; j++ ){ document.write(j) }    //11232123
    }
}

breack문

  • 반복문에서 breack문을 사용시 조건식과 상관없이 강제로 반복을 종료합니다.

continue문

  • 반복문에서 continue문 사용시 다음 조건식을 생략하고 그 다음 조건식으로 넘어갑니다.
