자료형(Data type)

  • 자료형(Data type)으로는 숫자, 문자열, 논리값, 특수값, 심벌, 객체, 배열, 함수 등이 있습니다.

숫자형(Number)

let 변수 = 숫자;
let 변수 = Number("문자형 숫자"); 
{
    let x = 100;                    //100
    let y = Number("200");    //200
}

문자형(String)

let 변수 = "사용할 문자나 숫자"; 
{
    let x = "javascript";    //javascript
    let y = "100";            //100
}

논리형(Boolean)

let 변수 = true or flalse;
let 변수 = Boollean(데이터); 
{
let a = true;                            //true
let b = flalse;                          //flalse
let c = 10 > 5;                         //true
let d = 10 < 5;                         //flalse

let e = Boollean("javascript");   //true
let f = Boollean(0);                  //flalse
let g = Boollean(null);              //flalse
let h = Boollean(undefined);    //flalse
let i = Boollean(" ");                  //flalse
}

특수값 : Null

  • null은 변수에 지정된 값이 null인 경우를 말합니다.
  • null은 변수 저장된 값을 비울떄 사용합니다.

특수값 : Undefined

  • Undefined은 변수에 값이 등록되기 전의 기본값을 말합니다.

특수값 : Typeof

  • typeof은 지정된 데이터 또는 변수에 저장된 자료형을 알고 싶을 때 사용합니다.
