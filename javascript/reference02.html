변수(Variables)

  • 변수란 변하는 데이터(값)을 저장할 수 있는 메모리 공간 입니다.
  • 변수에는 데이터가 오직 한 개만 저장됩니다.
  • 새로운 데이터가 들어오면 기존에 있던 데이터는 메모리 공간에서 지워지게 됩니다.
  • 변수에 저장할 수 있는 데이터의 종류로는 문자형, 숫자형, 논리형, 빈 데이터가 있습니다.

변수 선언

  • 변수를 선언할 때는 var 키워드를 변수명 앞에 붙입니다.
  • 변수명에는 한글을 사용할 수 없습니다.
  • 변수명에는 영문과 숫자 그리고 일부 특수 문자(_, $)만 포함할 수 있습니다.

변수(Variables) 실습해보기

<script>
    var box;
    box = 100;
    box = 30;
    document.write(box);
<script>

변수에 선언 시 주의 사항

  • 변수명 첫 글자로는 영문자, $, _만 올 수 있습니다.
  • 변수명 첫 글자 다음은 영문자, $, _, 숫자만 포함할 수 있습니다.
  • 변수명으로 예약어를 사용할 수 없습니다.
  • 변수명을 지을 때는 되도록 의미를 부여해 작성하는 것이 좋습니다.
  • 변수명을 사용할 떄느 대·소문자를 구분해야 합니다.

변수명

  • 자바스크립트에는 의미를 가진 단어들이 있기 때문에 사용자가 임의로 사용할 수 없는 단어들이 있습니다.
  • 키워드 : 자바스크립트에서 사용하는 단어
  • 식별자 : 사용자가 임의로 사용하는 단어

식별자를 만드는 규칙

  • 사용자가 사용할 수 있는 문자 : 알파벳(a~z, A~Z), 숫자(0~9), 밑줄(_), 달러기호($)
  • 첫 글자는 숫자를 사용할 수 없습니다.
  • 공백을 사용할 수 없습니다. : 밑줄(_) 사용가능, 하이픈(-) 사용안됩니다.
  • 특수기호를 사용할 수 없습니다. : 달러기호($)는 사용가능합니다.
  • 키워드를 사용할 수 없습니다.

단어를 조합할 때 규칙

  • 언더스코어 노테이션 : 중간에 '_'를 사용합니다.
    ex) color_top
  • 캐멜 노테이션 : 두번째 단어의 첫 글자를 대문자로 사용합니다.
    ex) colorTop
  • 파스칼 노테이션 : 모든 단어의 첫 글자를 대문자로 사용합니다.
    ex) ColorTop
